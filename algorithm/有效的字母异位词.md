# 题目
给定两个字符串 s 和 t ，编写一个函数来判断 t 是否是 s 的字母异位词。

示例:
```
输入: s = "anagram", t = "nagaram"
输出: true
```
# 思路
hash
```java{.line-numbers}
class Solution {
    public boolean isAnagram(String s, String t) {
        int[] dict = new int[26];
        for(int i = 0; i < s.length();++i){
            ++dict[s.charAt(i) - 'a'];
        }
        for(int i = 0; i < t.length();++i){
            --dict[t.charAt(i) - 'a'];
        }
        for(int i = 0; i < 26; ++i){
            if(dict[i] != 0){
                return false;
            }
        }
        return true;
    }
}
```
